# Demo Site — Documentation Skills Toolkit

> Chạy demo site để xem output thực tế khi dùng bộ toolkit.

## Quick Start

```bash
cd demo-site
pip install -r requirements.txt
mkdocs serve
# → Mở http://localhost:8000
```
